# Spentzos_Lab_Methylation2026

This repository contains the official code, statistical workflows, and predictive modeling pipelines for the manuscript: "Epigenetic patterns and methylation-based models for robust outcome prediction in osteosarcoma"

https://www.medrxiv.org/content/10.1101/2025.11.19.25340108v2

---

This repository serves as a comprehensive code reference to ensure the reproducibility of our findings. The analytical pipeline can be broken down into seven section, each of which roughly translates to a section within the manuscript:

* Section 1: Data Preprocessing
* Section 2: VMR Derivation and Exploration
* Section 3: Univariable Analysis
* Section 4: Supervised Analysis
* Section 5: Validation Cohort Analysis
* Section 6: Epigenetic Clock Analysis
* Section 7: Multi-Omics Comparative Analysis
